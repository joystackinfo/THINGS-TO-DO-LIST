# 📝 Things To Do List

A simple and clean to-do list app for keeping track of daily tasks.  

## ✨ Features
- Add and remove tasks  
- Mark tasks as done  
- Stay organized with ease  

## 🚀 How to Use
1. Open `index.html` in your browser  
2. Start adding your tasks!
